## 创建正则表达式

### 方式一：字面量

```javascript
var reg = /abc/;
```

### 方式二：构造函数

```javascript
var reg = new RegExp(/abc/);
var reg = new RegExp('abc');
```

## 检测方法

### test 方法

```javascript
// 检测字符串是否符合正则规则
var reg = /abc/;
reg.test('abc');  // true
reg.test('def');  // false
```

### exec 方法

```javascript
// 检索字符串中的匹配项
var str = 'hello';
var pattern = /o/;
pattern.exec(str);  // ['o', index: 4, ...]
```

## 边界符

| 符号 | 说明 | 示例 |
|------|------|------|
| `^` | 以某内容开头 | `/^abc/` |
| `$` | 以某内容结尾 | `/abc$/` |
| `^$` | 精确匹配 | `/^abc$/` |

```javascript
/^abc/;    // 以 abc 开头
/abc$/;    // 以 abc 结尾
/^abc$/;   // 精确匹配 abc
```

## 字符类

### 方括号

```javascript
/[abc]/;     // 匹配 a、b 或 c 中的任意一个
/[^abc]/;    // 匹配除了 a、b、c 之外的字符
/[a-z]/;     // 匹配 a 到 z 之间的任意小写字母
/[A-Z]/;     // 匹配 A 到 Z 之间的任意大写字母
/[0-9]/;     // 匹配 0 到 9 之间的任意数字
/[a-zA-Z0-9]/; // 匹配字母或数字
```

## 量词符

| 符号 | 说明 | 示例 |
|------|------|------|
| `*` | 出现 0 次或多次 | `/a*/` |
| `+` | 出现 1 次或多次 | `/a+/` |
| `?` | 出现 0 次或 1 次 | `/a?/` |
| `{n}` | 出现 n 次 | `/a{3}/` |
| `{n,}` | 出现 n 次或更多 | `/a{3,}/` |
| `{n,m}` | 出现 n 到 m 次 | `/a{3,6}/` |

```javascript
/a*/;      // a 出现 0 次或多次
/a+/;      // a 出现 1 次或多次
/a?/;      // a 出现 0 次或 1 次
/a{3}/;    // a 出现 3 次
/a{3,}/;   // a 出现 3 次或更多
/a{3,6}/;  // a 出现 3 到 6 次
```

## 预定义类

| 符号 | 说明 | 等价于 |
|------|------|--------|
| `\d` | 数字 | `[0-9]` |
| `\D` | 非数字 | `[^0-9]` |
| `\w` | 字母、数字、下划线 | `[a-zA-Z0-9_]` |
| `\W` | 非字母、数字、下划线 | `[^a-zA-Z0-9_]` |
| `\s` | 空格（换行、制表、空格） |  |
| `\S` | 非空格 |  |

```javascript
// | 表示或
var reg = /^(aaa)|(bbb)$/;
```

## 常用正则示例

### 座机号码

```javascript
var reg = /^\d{3}-\d{8}|\d{4}-\d{7}$/;
```

### 手机号码

```javascript
var reg = /^1[3-9]\d{9}$/;
```

### QQ 号码

```javascript
var reg = /^[1-9]\d{4,}$/;
```

### 中文

```javascript
var reg = /[\u4e00-\u9fa5]/;
```

### 邮箱

```javascript
var reg = /^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/;
```

### 身份证号

```javascript
var reg = /^\d{15}|\d{18}$/;
```

## 匹配模式

### 忽略大小写

```javascript
/abc/i;  // 忽略大小写
```

### 全局匹配

```javascript
/abc/g;  // 匹配所有满足条件的字符
```

### 组合使用

```javascript
/abc/ig;  // 忽略大小写 + 全局匹配
```

## 分组

```javascript
// () 包起来的内容表示一个分组
var reg = /{{ ([a-zA-Z]+) }}/;
var str = '{{name}}';
var result = reg.exec(str);
// result[0] = '{{name}}'
// result[1] = 'name'
```

## 括号总结

| 括号 | 作用 |
|------|------|
| `{}` | 量词符，表示重复次数 |
| `[]` | 字符集合，匹配方括号中的任意字符 |
| `()` | 分组，表示优先级 |

## 正则与字符串方法

### match

```javascript
// 返回匹配的数组
str.match(/abc/g);
```

### replace

```javascript
// 替换匹配的内容
str.replace(/abc/g, 'def');
```

### search

```javascript
// 返回匹配的位置索引
str.search(/abc/);
```

### split

```javascript
// 根据正则分割字符串
str.split(/\s+/);
```
